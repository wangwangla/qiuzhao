## Liunx文件管理和目录配置

## 目录与路径

1. 相对路径和绝对路径

2. 目录相关操作

   ```
   .当前目录
   ..上一层目录
   -代表前一个目录
   ~主目录
   ~kw主目录为kw的家目录
   ```

3. 命令相关命令

   ```
   cd：切换目录
   pwd:显示当前目录位置
   mkdir:创建目录 
   rmdir:删除目录
   ```

   pwd -P:显示绝对路径，不加是相对的连接目录

   mkdir:m是带有权限的创建

   p:是递归起来创建

   ```
   mkdir -m 777 test
   ```

   删除也可以带有这个参数，就是递归删除。

4. 环境变量

   每个用户都可以通过这个命令，看出自己的环境变量，举个例子，一个文件执行的路径是/test/test,加入你的环境变量没有，你就没有权限，有这个目录的环境变量就可以执行。

5. ls命令

   - a：全部文件，
   - A
   - d仅仅是目录
   - f列出不排序
   - h:比较易读的方式
   - i:node列出

6. cp rm mv

   - cp : 

     - i :如果文件存在，是否将其覆盖。
     - r： 

     复制到当前目录 cp xx .

     复制如果不加任何参数，那么文件的属性和权限都会改变。

     需要有r权限

   - rm:删除文件

     - -f:强制删除
     - -i:互动模式，删除前询问
     - -r:递归删除

   - mv:移动文件

     - -f:强制
     - -i:询问
     - -u:存在。保存新的

7. 查询内容

   cat：查看第一行

   tac:最后开始

   more:一页一页显示

   less:他可以翻页

   head:只看开头

   tail:只看结尾

   od:以二进制的方式读取。

   - 查看内容
     - cat /etc/isuse
   - 加行号
     - cat -n /etc/tets

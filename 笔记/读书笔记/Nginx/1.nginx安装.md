## nginx在window下安装
```
    下载
        下载：http://nginx.org/
        下载windows和linux
    安装    
        windows下安装
        conf：是配置文件    
        contrib:目录存放一些实用工具   
        docs:目录存放文档   
        html：目录存放网站默认目录   
        logs：日志
        temp:目录存放临时文件
    启动
        双击nginx.exe,使用cmd查看命令进程
        默认监听80
        执行netstat -ano查看80是否被占用
        netstat -ano | findstr 80
    停止
        到达目录下，执行nginx.exe -s stop
    修改端口
        F:\nginx-1.12.2\conf\nginx.conf
        server {
            listen       8088;
            server_name  localhost;
        }

```
## nginx在linux下安装
```
    1.安装依赖
      gcc：编译c语言
      gcc-c++：编译c++
      wget:从网络下载资源

      yum -y install gcc gcc-c++ wget
    2.先将原来的的卸载
      find / -name nginx* 先将其找出来
      rm -rf file 
    3.获取nginx
    4.解压
      tar -zxvf nginx-XX
    5.cd nginx
    6.编译安装nginx
      （1）安装依赖
           yum -y install pcre-devel openssl-devel
      （2）编译选项
            ./configure --prefix=/usr/local/nginx --with-http_ssl_module
            对于安装进行配置  设置安装目录  设置是否允许http_ssl_module相关功能
            暂时先安装这几个，如果需要其他的就是有--with-进行添加，并重新进行编译
      （3）编译安装
            make&&make install
    7.启动
      cd /usr/local/nginx/sbin
      ./nginx

      查看是否执行成功，如果占用
        ps aux | grep nginx
        ps -ef | grep nginx查看端口
        kill -quit 4730  杀掉他
      未占用打开浏览器 
       http://10.114.114.201/
    8.停止
      ./nginx -s stop  直接停止
      ./nginx -s quit  任务结束，在停止
      kill nginx进程
      kiladll nginx

    9.其他指令
      nginx -s reload 重启
      nginx -s reopen重新打开日志
      nginx -c /目录/nginx.conf 指定目录下的配置启动
      nginx -t 检查当前配置文件是否正确
      nginx -v 显示版本
      nginx -V 显示版本以及编译选项

    10.防火墙设置【这个一般是找到防火墙的那个文件，然后加入，在重新启动】                                      
       iptables -I input -p tcp --dport 80 -j ACCEPT
       -I INPUT:表示插入一条规则
       -p tcp :指定数据包的协议
       -dport 80：开放的端口
       -j ACCEPT:指定数据包的处理操作
    11.为了可以愉快的玩耍，将其加入环境变量
       [root@localhost sbin]# echo $PATH
       /usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/usr/local/nginx/sbin:/usr/local/jdk1.8.  0_111/bin:/usr/local/node/bin:/root/bin
      sbin：创建管理员执行的指令
      bin:执行一般的
      usr/bin 和 /usr/sbin放置一些工具
      usr/local/bin 和 /usr/local/sbin放置一些用户的工具，因此创建软连接
      ln -s /usr/local/nginx/sbin/nginx /usr/local/sbin/nginx
      软连接创建后就可以进行操作了
      nginx -s quit
      nginx
      nginx -s reload
    12.添加到服务
       /etc/init.d在这个目录下，增加一个nginx的shell就可以了。这样就可以通过服务来开启了。
    13.开机自启
       通过chkconfig完成开机自启

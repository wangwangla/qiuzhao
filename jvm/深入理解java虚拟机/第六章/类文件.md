# 类文件
```

    java在各种平台上都是指向字节码文件，所以他是跨平台的，一个平台只需要安装对应的java虚拟机就可以了。
    无论什么语言只要最终编译成了字节码文件就可以在平台进行运行。

```
### class类文件
```
    
    class文件是一个以8位字节码为基础的单位的二进制流，各个数据项目严格按照紧凑低排列在class文件之中，中间没有任何分隔符，如果大于8个字节，就会形成大的在前，然后分割成8字节的进行存储。




```
### 类加载器
```

    类加载阶段，通过一个类的权限定名来获取描述次类的二进制字节流，放到java虚拟机外部去实现，以便应用程序自己决定如何去获取所需要的类，这个过程就是累加载器的过程
